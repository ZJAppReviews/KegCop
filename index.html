About

KegCop is open source iOS application that monitors a kegerator by having drinkers create user accounts associated with the kegerator. The admin of the kegerator will login with the root account, and then issue credits to the users of the kegerator. The source code for the application can be found here, https://github.com/ipatch/KegCop
